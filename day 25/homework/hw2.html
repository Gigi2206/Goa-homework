CSS-ის წესი (CSS Rule)

CSS-ის წესი განსაზღვრავს, თუ როგორ გამოიყურება HTML ელემენტი. იგი შედგება სელექტორისა და დეკლარაციის ბლოკისგან.

            
                <style>
                  h2 { /* სელექტორი - h2 ელემენტი */
                    color: green; /* დეკლარაცია */
                    text-align: center; /* კიდევ ერთი დეკლარაცია */
                  }

                  .important-paragraph { /* სელექტორი - class="important-paragraph" მქონე ელემენტები */
                    font-size: 18px; /* დეკლარაცია */
                    font-weight: bold; /* დეკლარაცია */
                  }
                </style>

ეს არის h2 ელემენტი, რომელსაც CSS-ის წესი აქვს გამოყენებული.

ეს არის პარაგრაფი class-ით "important-paragraph", რომელსაც CSS-ის წესი აქვს გამოყენებული.

დეკლარაცია (Declaration)

დეკლარაცია განსაზღვრავს კონკრეტულ სტილს ელემენტისთვის. იგი შედგება კუთვნილებისა და მისი მნიშვნელობისგან.

            
                color: green; /* დეკლარაცია */
                text-align: center; /* დეკლარაცია */
                font-size: 18px; /* დეკლარაცია */
                font-weight: bold; /* დეკლარაცია */

ზემოთ მოყვანილია დეკლარაციების მაგალითები.

კუთვნილება (Property)

კუთვნილება არის CSS-ის სტილის ატრიბუტი, რომელსაც იყენებთ ელემენტის გარეგნობის შესაცვლელად.

  • color - ტექსტის ფერი
  • text-align - ტექსტის განლაგება
  • font-size - ტექსტის ზომა
  • font-weight - ტექსტის სისქე
  • background-color - ფონის ფერი

მნიშვნელობა (Value)

მნიშვნელობა განსაზღვრავს, თუ რა სტილი მიენიჭება კონკრეტულ CSS კუთვნილებას.

  • ფერის მნიშვნელობები: green, blue, #ff0000
  • ზომის მნიშვნელობები: 18px, 1.5em, 100%
  • საკვანძო სიტყვები: center, bold, italic
